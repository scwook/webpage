EPICS Configuration Guide

git에서 EPICS script를 받기위해 우선 git을 설치한다.

scwook@scwook:~$ sudo aptitude install git

git으로 부터 EPICS script를 받운 받는다.

scwook@scwook:~$ git clone https://github.com/jeonghanlee/scripts_for_epics.git
Cloning into 'scripts_for_epics'...
remote: Counting objects: 99, done.
remote: Total 99 (delta 0), reused 0 (delta 0)
Unpacking objects: 100% (99/99), done.

EPICS 설치 script를 실행하기 전 필요한 library를 설치한다.

scwook@scwook:~$ sudo aptitude install libreadline-dev g++ libxt-dev lesstif2-dev x11proto-print-dev libxmu-headers libxp-dev libxmu-dev
libxmu6  libxpm-dev libxmuu-dev libxmuu1 libxmu6

EPICS script를 실행한다.

scwook@scwook:~$ cd scripts_for_epics
scwook@scwook:~/scripts_for_epics$ bash epics_default_installation.sh

EPICS 설치를 마쳤으면 siteLibs과 siteApps를 git에서 다운 받는다.

scwook@scwook:~/epics/R3.14.12.4$ git clone https://github.com/RaonControl/siteLibs_test siteLibs
