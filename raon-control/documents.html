EPICS is a set of software tools and applications for distributed control systems.

Step1. Installation

EPICS can download on the EPICS download page

$ wget https://epics-controls.org/download/base/base-3.15.7.tar.gz

$ tar xvf base-3.15.7.tar.gz

$ cd base-3.15.7

base-3.15.7$ make

Step2. Environment Configuration

Before use the EPICS, we need a configuration of environment path

$ export PATH=$PATH:/home/scwook/base-3.15.7/bin

Step3. Example

Now, we can use the EPICS. Let's start get some example

$ mkdir myExample

$ cd myExample

myExample$ makeBaseApp -t example example

myExample$ makeBaseApp -i -t example example

myExample$ make
